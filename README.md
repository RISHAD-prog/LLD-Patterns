
# 📐 LLD Design Patterns – Codemia Problems

This repository contains my solutions to Low-Level Design (LLD) problems sourced from Codemia. The primary focus is on applying design patterns, object-oriented design principles, and writing clean, maintainable, and scalable code.

---

## 🚀 Objectives

- Strengthen problem-solving skills in Low-Level Design
- Practice and implement industry-standard design patterns
- Improve code structure, readability, and maintainability
- Build a strong foundation for system design interviews

---

## 🧠 Design Patterns Covered

This repository includes implementations of commonly used design patterns such as:

- Singleton
- Factory
- Strategy
- Observer
- Builder
- Adapter
- Decorator
- Command
- State
- Chain of Responsibility

> More patterns will be added as I solve additional problems.
